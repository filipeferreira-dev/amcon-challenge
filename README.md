# Tecnologias utilizadas

Temos duas aplicações uma api e uma aplicação web e uma api .
Tecnologias utilizadas:
- .NET 7
- C#
- SQL SERVER

# Como executar o projeto

Pré-requisitos: Docker

Na pasta raiz do projeto execute o comando abaixo:
```bash
docker-compose up -d
```

Depois dos container iniciarem você pode acessar a aplicação no seguintes caminhos:

Web: `http://localhost:5001`  

Api: `http://localhost:5000/swagger/`
